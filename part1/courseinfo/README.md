# Course Information

This is the solution for exercises 1.1-1.5 of Full Stack Open Part 1.

## Exercises

This application demonstrates:
- **1.1-1.2**: Breaking down the application into components (Header, Content, Total)
- **1.3-1.4**: Refactoring to use objects for course data
- **1.5**: Further refactoring with improved data structures

## Running the Application

```bash
npm install
npm run dev
```

The application will be available at http://localhost:5173/

## Building for Production

```bash
npm run build
```

## Linting

```bash
npm run lint
```
