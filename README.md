# fullstack-open
Course material for Full Stack Open by University of Helsinki.
https://fullstackopen.com/en/

## Course Overview
This repository contains exercises and projects covering modern web development with JavaScript, React, Node.js, and more. The course is designed to provide hands-on experience in building full-stack applications.

## Setup
To run the exercises locally, clone the repository and follow the instructions in each exercise folder.
