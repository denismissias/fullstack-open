# Full Stack Open

This repository contains my solutions to the exercises from the [Full Stack Open](https://fullstackopen.com/) course by the University of Helsinki.

## About the Course

Full Stack Open is a comprehensive course on modern web development with JavaScript. The course covers:
- React
- Redux
- Node.js
- MongoDB
- GraphQL
- TypeScript
- React Native
- CI/CD
- Containers
- Relational databases

## Course Structure

### Part 0: Fundamentals of Web apps
Introduction to web applications and HTTP protocol.

### Part 1: Introduction to React
Introduction to React library, JavaScript, and state management.

### Part 2: Communicating with server
Working with forms, fetching data from a server, and rendering collections.

### Part 3: Programming a server with NodeJS and Express
Building a backend with Node.js and Express, REST APIs, and deploying to the internet.

### Part 4: Testing Express servers, user administration
Writing unit and integration tests for backends, user authentication and authorization.

### Part 5: Testing React apps
Testing React applications with Jest and Cypress, token-based authentication.

### Part 6: Advanced state management
Complex state management with Redux and React Query.

### Part 7: React router, custom hooks, styling app with CSS and webpack
React Router, custom hooks, and more advanced styling approaches.

### Part 8: GraphQL
GraphQL and how to use it with React and Node.js.

### Part 9: TypeScript
TypeScript - types for JavaScript.

### Part 10: React Native
Building mobile applications with React Native.

### Part 11: CI/CD
Introduction to Continuous Integration and Continuous Delivery.

### Part 12: Containers
Introduction to containers and container orchestration with Docker.

### Part 13: Using relational databases
Using relational databases with Node.js using PostgreSQL and Sequelize.
